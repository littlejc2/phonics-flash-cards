# 🎛️ AI模型选择指南

## 🎉 新功能：智能模型选择

现在你可以为每个AI提供商选择不同的模型，以获得最适合你需求的AI体验！

## 📍 在哪里选择模型

### 1. **API配置页面** (推荐)
- 访问：http://localhost:8080/api-setup
- 在每个提供商的标签页中选择模型
- 显示详细的模型信息和特性

### 2. **主页面**
- 在单词输入表单中选择提供商和模型
- 实时切换，立即生效

### 3. **测试页面**
- 访问：http://localhost:8080/ai-test
- 测试不同模型的性能表现

## 🔄 如何切换模型

### 方法1：配置页面设置
1. 访问 http://localhost:8080/api-setup
2. 选择提供商标签页
3. 在"模型选择"下拉菜单中选择
4. 点击"保存配置"

### 方法2：主页面实时切换
1. 在主页面的表单中
2. 先选择AI提供商
3. 再选择具体模型
4. 立即生效，无需保存

### 方法3：localStorage直接设置
```javascript
// 设置Gemini模型
localStorage.setItem('gemini_model', 'gemini-1.5-pro-latest');

// 设置DeepSeek模型
localStorage.setItem('deepseek_model', 'deepseek-coder');

// 刷新页面生效
window.location.reload();
```

## 🔧 技术实现

### 配置存储
- 模型选择保存在 `localStorage`
- 键名格式：`{provider}_model`
- 例如：`gemini_model`, `deepseek_model`

### 运行时获取
```typescript
import { getCurrentModel } from '@/config/gemini';

// 获取当前提供商的模型
const model = getCurrentModel('gemini');

// 获取指定提供商的模型
const deepseekModel = getCurrentModel('deepseek');
```

### API调用
- 前端：自动使用选择的模型
- 后端：Supabase函数支持模型参数
- 存储：配置持久化保存
